# MuQiuEmacs
> 一份比较基础快速的Emacs配置
> 最近更新: 2021-1-15
> 更新信息: window-numbering org-mode 白色主题

## 安装

```Shell
cd ~
git clone https://gitee.com/muqiu-han/emacs.d .emacs.d
```
## Treemacs
__C-x C-t__ 打开/关闭treemacs

## org-mode
__C-c C-h__ 生成当前文件的html文件
__C-c C-m__ 生成当前文件的markdown文件

## window-numbering
__M-(1~9)__ 跳转到某个窗口
